# SuConnect – University Event Organization Platform

Sabancı University – CS310 Mobile Application Development  
Umut Köprülü  

---

## Overview

SuConnect is a cross-platform mobile application developed to simplify event organization within Sabancı University.  

The application enables students to create events, join activities, and connect with others for academic, social, and sports collaborations.

The system focuses on structured user management, real-time event handling, and scalable database integration.

---

## Core Features

- User Authentication (Firebase Authentication)
- Student Profile Management
- Event Creation
- Event Participation System
- Event Listing & Filtering
- Real-time Database Updates
- Cross-platform deployment (Android, iOS, Web)

Each student profile includes:

- Student ID
- Name & Surname
- Email
- Created Events
- Joined Events

## Architecture

- Frontend built with Flutter (modular UI structure)
- Backend powered by Firebase
- Firestore-based event and user data storage
- Authentication-controlled access flow
- Real-time event state synchronization

## Development Process

The project followed an iterative development approach:

1. Requirement analysis and feature definition
2. UI wireframe design
3. Firebase authentication integration
4. Database schema design (users & events collections)
5. Event CRUD operations implementation
6. Real-time event listing and updates
7. Cross-platform testing and debugging
